# world-cup-intelligence-center
Plataforma corporativa de inteligência esportiva construída sobre n8n como orquestrador principal. Coleta dados em tempo real da Copa do Mundo 2026, processa eventos ao vivo, gera análises com GPT-4o, monitora sentimento social e distribui alertas via múltiplos canais. Stack: n8n Queue Mode, PostgreSQL + TimescaleDB, Redis, Prometheus, Grafana.
